# **Glowing Geometric Donut**

## **Description**

I created a Glowing Geometric Donut animation using Python's Turtle module. The program draws a spinning, glowing donut made of circles with changing colors, creating a visually appealing geometric effect. This project helped me understand loops, colors, and basic animation in Python.

## **Features**

- Draws a rotating donut shape using Turtle graphics.
- Uses rainbow colors that gradually change to give a glowing effect.
- Creates a smooth animation by repeatedly drawing overlapping circles.
- Demonstrates how loops and trigonometry can create geometric patterns.
- Interactive visual effect that looks like a glowing, spinning donut.

## **How It Works**

- I used Python’s turtle module to draw on a canvas and colorsys to create smooth rainbow color changes.
- A for loop runs 180 times to draw multiple circles while gradually changing the hue (h) for the glowing effect.
- Commands like circle(), forward(), left(), and right() are used to move the pen and create the donut pattern.
- Each iteration slightly rotates the pen and moves it forward, overlapping circles to create the spinning geometric donut effect.
- The background is black to make the glowing colors stand out.
